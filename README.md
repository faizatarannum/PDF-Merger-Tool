# PDF Merger Tool (Python GUI)

A simple GUI-based PDF merger tool built using Python, Tkinter, and PyPDF2.

## Features
- Add multiple PDF files
- Rearrange order (Move Up / Move Down)
- Remove selected files
- Merge PDFs into one
- Simple user-friendly interface

## Technologies Used
- Python
- Tkinter
- PyPDF2

## How to Run
pip install PyPDF2
python merger.py
